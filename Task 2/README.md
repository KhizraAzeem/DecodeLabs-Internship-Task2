# 🎓 E-Learn Hub

E-Learn Hub is a modern, responsive educational landing page built using **HTML5** and **CSS3**. It is designed for online learning platforms, educational institutes, course-selling websites, and e-learning startups.

The website features a professional hero section, course showcase cards, statistics section, feature highlights, and a responsive layout that works across desktop, tablet, and mobile devices.

---

## 📌 Features

### ✅ Responsive Navigation Bar
- Sticky navigation menu
- Smooth hover effects
- Mobile-friendly layout

### ✅ Hero Section
- Full-screen background image
- Attractive call-to-action button
- Professional overlay effect

### ✅ Popular Courses Section
- Web Development Course
- Graphic Design Course
- Programming Course
- Hover animation effects

### ✅ Statistics Section
Displays platform achievements such as:
- Students enrolled
- Available courses
- Instructors
- Success rate

### ✅ Why Choose Us Section
Highlights:
- Expert Teachers
- Flexible Learning
- Certificates

### ✅ Responsive Design
Optimized for:
- Desktop
- Tablet
- Mobile Devices
